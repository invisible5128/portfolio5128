FAST SCAN APP

Fast Scan is QR code scanner built using Flutter.

About App

Android App built using Flutter. The App can be used to scan the QR code.

APP features

  • Scans QR Code and keeps track of previous scanned QR code texts.
  • App deployed in Google Play Store in Beta Stage.
  • Uses Offline Storage to keep track of previous scanned QR codes.
